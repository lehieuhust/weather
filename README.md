# `weather`
weather tool.

## Usage

```
Usage: weather [options] [city name[,state code][,country code]]

Options:
    -p, --location-provider 0 to 3
                        Location provider
    -f, --full-info     Full weather information
    -s, --silent        Silent mode
    -v, --version       Print program version
    -h, --help          Print this help menu
```
